# HR Database Project

This project connects to a MySQL database and performs various SQL queries to retrieve and analyze HR data.

## Features
- Retrieve employee details.
- Analyze salaries, departments, and job history.
- Use Python and MySQL for data manipulation.

## Requirements
- Python 3.x
- `pymysql` library
- MySQL database
